# Handbook Feedback / 手册反馈

> This directory contains feedback, questions, and suggestions for the AI × Web3 School Handbook.
> 
> 本目录包含对 AI × Web3 School 手册的反馈、问题和建议。

## 📝 How to Provide Feedback / 如何提供反馈

When you encounter issues while learning from the Handbook, create a new markdown file here with:

当你在学习手册时遇到问题，请在此创建一个新的 markdown 文件，包含：

1. **Handbook page link / 手册页面链接**
2. **Issue description / 问题描述**
3. **Suggested improvement / 改进建议**
4. **Context / 背景信息** (optional)

## 📂 File Naming Convention / 文件命名规范

Use the format: `YYYY-MM-DD-topic-name.md`

Example: `2026-05-18-llm-context-window.md`

## 🎯 Types of Feedback / 反馈类型

- **Typos / 错别字**: Spelling or grammar errors
- **Unclear Concepts / 概念不清**: Confusing explanations
- **Missing Information / 信息缺失**: Topics that need more detail
- **Outdated Content / 过期内容**: Information that needs updating
- **Structure Suggestions / 结构建议**: Better organization ideas
- **Code Examples / 代码示例**: Request for more examples
- **Translation Issues / 翻译问题**: Chinese-English translation improvements

