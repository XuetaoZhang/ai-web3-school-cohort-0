# DAO 提案研究 Agent

基于 AI × Web3 School Handbook 的标准架构实现的 DAO 提案研究助手。

## 📋 功能概述

这是一个**只读分析型 Agent**，帮助 DAO 成员研究提案并做出明智的投票决策。

### Agent 能执行的操作

1. ✅ **读取提案正文** - 获取提案的完整内容
2. ✅ **检索论坛讨论** - 查找社区对提案的讨论
3. ✅ **总结支持和反对理由** - 分析正反方观点
4. ✅ **标记缺失信息** - 指出需要补充的信息
5. ✅ **输出投票前检查清单** - 生成决策辅助清单

### Agent 不会做的事

❌ **不会直接投票** - Agent 只提供分析，不代替用户做决策

### 输出要求

Agent 必须在输出中明确：
- 用了哪些来源
- 哪些结论证据不足
- 是否发现治理或资金风险
- 如果用户要投票，还需要人工检查什么

## 🔧 技术实现

### 核心组件

1. **System Prompt** - 定义 Agent 的角色和行为
2. **Tools** - 5 个专业工具函数
3. **LLM** - DeepSeek API（支持 Function Calling）
4. **Agentic Loop** - 多轮对话和工具调用循环

### 与 Web3 AI Assistant 的对比

| 特性 | Web3 AI Assistant | DAO Research Agent |
|------|-------------------|-------------------|
| **架构** | 简单工具调用 | 标准 Agentic Loop |
| **工作流程** | 单次工具调用 | 多步骤流程 |
| **System Prompt** | 简单角色定义 | 详细工作流程 |
| **输出要求** | JSON 格式 | 结构化报告 + 检查清单 |
| **安全性** | 只读分析 | 只读 + 明确风险提示 |
