# AI Agent 实验报告

## 📋 实验信息

- **实验日期**: 2026-05-19
- **实验主题**: AI Agent 与 Tool Use
- **使用模型**: DeepSeek Chat
- **实验目标**: 理解 Agent 的工作原理和 Tool Use 机制

---

## 🎯 实验目标

1. 理解 AI Agent 的核心概念
2. 掌握 Tool Use (Function Calling) 的工作流程
3. 实现一个能够调用多个工具的 Agent
4. 观察 Agent 的决策过程和多步推理能力

---

## 🔧 实验环境

### 工具列表

| 工具名称 | 功能 | 参数 |
|---------|------|------|
| calculator | 计算数学表达式 | expression: str |
| get_current_time | 获取当前时间 | timezone: str (可选) |
| get_weather | 查询城市天气 | city: str |
| search_knowledge | 搜索知识库 | query: str |

### 技术栈

- Python 3.x
- OpenAI SDK (兼容 DeepSeek API)
- DeepSeek Chat 模型

### 测试案例 1: 简单工具调用

#### 案例 1.1: 数学计算

**用户输入**: "计算 (123 + 456) * 789 的结果"

**Agent 行为**:
```
1. 识别需要计算器工具
2. 调用 calculator("(123 + 456) * 789")
3. 获取结果: 456831
4. 返回答案
```

**结果**: ✅ 成功

**观察**:
- Agent 正确识别了计算需求
- 工具调用参数正确
- 结果整合自然

### 测试案例 2: 多轮对话与复杂推理

#### 案例 2.1: 多步计算

**用户输入**: "先计算 100 * 50，然后用结果除以 25"

**Agent 行为**:
```
第 1 轮:
  - 调用 calculator("100 * 50")
  - 结果: 5000

第 2 轮:
  - 调用 calculator("5000 / 25")
  - 结果: 200

最终答案: "结果是 200"
```

**结果**: ✅ 成功 / ❌ 失败

**观察**:
- [ ] Agent 是否理解了多步任务？
- [ ] 是否正确分解了步骤？
- [ ] 中间结果是否正确传递？
